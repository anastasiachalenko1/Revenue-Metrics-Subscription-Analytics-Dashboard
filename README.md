# Revenue-Metrics-Subscription-Analytics-Dashboard
Built a SaaS analytics dashboard in Tableau using PostgreSQL to track MRR, churn, LTV, and expansion/contraction revenue. Wrote optimized SQL queries with window functions to calculate core subscription KPIs, with filters by date, language, and age group.

Tools: SQL (PostgreSQL), Tableau Public, Data modelling, Data visualization

Overview: Built comprehensive subscription analytics dashboard processing 50,000+ records to visualize revenue dynamics, customer behavior, and growth patterns for SaaS business model.

Responsibilities: Wrote SQL queries to extract and aggregate data, calculated core SaaS metrics and built a structured dataset for analysis. Designed the Tableau dashboard with filters by date, user language, and age, and created visualizations to show month-to-month changes in revenue and paid users, following data visualization best practices.

Result: Translated business questions around revenue sustainability and retention into structured analytical models and visualization layers. Delivered an executive-ready dashboard consolidating 6 core subscription KPIs into a unified decision-support view. Highlighted revenue risk exposure through churn-revenue gap analysis and contraction MRR tracking. Reduced query execution time by over 92% (from 30 to 2.3 seconds), improving dashboard responsiveness and analytical efficiency.
